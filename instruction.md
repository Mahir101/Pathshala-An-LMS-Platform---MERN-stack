To run server:
1. Go to \server
2. Delete json.lock file
3. Enter 'npm install' command in CMD
4. Then enter 'npm start' command in CMD to run server at localhost:8000

To run client:
1. Go to \client
2. Delete json.lock file
3. Enter 'npm install' command in CMD
4. Then enter 'npm run dev' command in CMD to run client at localhost:3000

Node version: 16.2.0
